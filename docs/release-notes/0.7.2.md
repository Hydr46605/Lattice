## Fixed

- Made the Paper bootstrap resilient when Paper reaches `onEnable()` before a Lattice runtime was created in `onLoad()`.
- Deferred Paper UI listener registration until the plugin is enabled, avoiding `IllegalPluginAccessException` during plugin loading.
- Regenerated the standalone plugin descriptor when the Lattice version changes so the packaged Paper metadata stays accurate.

## Compatibility

- No plugin authoring API changes are required.
- Existing plugins extending `LatticePaperPlugin` keep the same entrypoint shape.
